# Github-Issues-Tracker

1. What is the difference between var, let, and const?

Ans. var is old and function-scoped, 
let is block-scoped and can change,
const is block-scoped and fixed

2. What is the spread operator (...)?

Ans. spreads elements of an array or properties of an object.

3. What is the difference between map(), filter(), and forEach()?

Ans. map() returns a new array, filter() returns a filtered array, forEach() just loops without returning.

4. What is an arrow function?

Ans. A short way to write a function using ()=> {}.

5. What are template literals?

Ans. Use ${} inside backticks `` to include variables in strings.
